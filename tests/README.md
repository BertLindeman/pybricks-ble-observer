### Pybricks test

This folder contains a few pybricks programs that can be used to run on a LEGO hub running pybricks firmware

| program name | Description |
|---|---|
| ble_broadcast_counter.py | send a counter incremented by one |
| ble_broadcast_channel_255.py | send various data types via channel 255 |
| ble_broadcast_all_ch.py | Use all channels to broadcast |
